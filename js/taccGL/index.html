NOTE This is BETA Software and known to contain bugs. It is not implemented completely and will probably not run on all browsers and client computers.

The library requires a client/browser with decent performance supporting HTML 5 canvas. If HTML 5 canvas is not present or if the library detects that the performance is not sufficient, it disables itself in an effort to minimize showing slow, choppy or jumpy animations, or to even overload the client. The idea is to keep the underlying HTML pages operational, if necessary without the taccGL ™ animations. See here for details.
